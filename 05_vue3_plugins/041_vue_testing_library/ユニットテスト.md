# ユニットテスト（Jest と vue-test-utils）

関数やストアやコンポーネントの単体テストを Jest と Vue-test-utils を組み合わせて行うことができる。js の関数は良いとして、Vue3 のコンポーネントのテストについてはまだあまり情報が出回っていないようだ。
